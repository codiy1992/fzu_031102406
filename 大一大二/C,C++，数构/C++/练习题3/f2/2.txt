定义一个时间类Time,内有数据成员hour,minute,second，另有成员函数：构造函数用于初始化数据成员，输出函数，自减运算符重载“--”的重载（包括前置和后置）。编写主函数：输入一个时间，计算并输出前一秒的时间。
输入格式： 测试输入包含若干测试用例，每个测试用例占一行。当读入0  0  0时输入结束，相应的结果不要输出。
输入样例：
6 3 5
20 13 0
7 0 0
0 0 0
输出样例：
predecrement=06:03:05,postdecrement=06:03:04L
predecrement=20:13:00,postdecrement=20:12:59L 
predecrement=07:00:00,postdecrement=06:59:59L
